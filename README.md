# Fuzzy Object Following e-puck Robot (Webots)

This project is a final assignment for **Sistem Kendali (Control Systems) RE 305**.  
It uses a **fuzzy logic controller** on the e-puck robot in Webots.  
The robot **follows a colored object** and **avoids obstacles** using its camera and proximity sensors.

## Requirements

- Python 3.8 or newer  
- OpenCV 4.x  
- NumPy  
- scikit-fuzzy  

## Build & Run

1. **Clone Repository**

       git clone https://github.com/ichsanyudika12/re305_webots_fuzzy_tracker.git
       cd re305_webots_fuzzy_tracker

2. **Open the world file**

       worlds/world.wbt

3. **Set the e-puck robot controller**

       tracking.py
